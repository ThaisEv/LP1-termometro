# LP1-termometro

<p>Este repositório contém a resolução da atividade proposta utilizando a linguagem C.</p>

<h2>Como compilar</h2>
<p>Use o compilador GCC no terminal:</p>

<pre>
gcc arquivo.c -o programa
</pre>

<h2>Como executar</h2>
<p>Após compilar, execute:</p>

<pre>
./programa
</pre>

<p>No Windows também pode executar com:</p>

<pre>
programa.exe
</pre>
